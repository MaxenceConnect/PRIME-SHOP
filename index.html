💳 Configuration des paiements

Configure ton lien PayPal pour encaisser de vrais paiements.

📋 Comment créer un lien PayPal :

  1. Crée un compte sur paypal.com
  2. Active ton lien PayPal.Me (paypal.com/paypalme)
  3. Colle l'URL ci-dessus — le montant sera ajouté automatiquement

💡 Le paiement par carte bancaire (Stripe) pourra être ajouté plus tard. Comme tu as moins de 18 ans, Stripe demandera qu'un parent ou tuteur soit déclaré titulaire du compte pour recevoir les paiements — utile à garder en tête si tu veux l'activer un jour.

📧 Notification de commande

Quand un client passe commande, un email t'est automatiquement envoyé avec les liens des produits à acheter chez le vendeur.

📋 Comment configurer EmailJS :

  1. Créer un compte gratuit sur emailjs.com
  2. Ajouter un service email (Gmail, Outlook…)
  3. Créer un template avec les variables : {{product_name}}, {{seller_links}}, {{client_name}}, {{client_address}}, {{client_email}}, {{quantity}}, {{total}}
  4. Copier vos IDs ci-dessus
